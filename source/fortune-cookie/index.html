User Guide

Step 1: Using the Fortune Cookie

  • Click the fortune cookie image to crack it open.
  • Read the fortune displayed on the paper inside the cookie.
  • Click the "New cookie" button to generate another fortune.
  • Use the voice feature to hear the fortune read aloud if desired.

Step 2: Additional Options

  • Use the "Skip animation" button to bypass the cookie cracking animation.
  • Change the voice settings using the dropdown menu in the footer.
